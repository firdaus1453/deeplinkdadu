# deeplinkdadu
Documentation for scheme deeplink dadu apps  

### For deeplink callback Gojek
BASE_URL = shober://gits.app/  
type = gojek  
result = success or failure  
idorder = id order, ex : 97d136e3-978d-4882-a06d-4785cc83a600
